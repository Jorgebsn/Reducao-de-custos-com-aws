# RELATÓRIO DE IMPLEMENTAÇÃO DE SERVIÇOS AWS

Data: 23/04/2026
Empresa: Abstergo Industries 
Responsavel: Jorge Batista dos Santos Neto

## Introdução
Este relatório apresenta o processo de implementação de ferramentas na empresa Abstergo Industries, realizado por Jorge Batista dos Santos Neto. O objetivo do projeto foi elencar 3 serviços AWS, com a finalidade de realizar diminuição de custos imediatos.

## Descrição do Projeto
O projeto de implementação de ferramentas foi dividido em 3 etapas, cada uma com seus objetivos especi­ficos. A seguir, serão descritas as etapas do projeto:

Etapa 1: 
- AWS Lambda

- Eliminar custos com servidores ociosos, executando código apenas quando necessário.

- A empresa pode utilizar o AWS Lambda para processar arquivos ou executar rotinas automáticas (como envio de relatórios), pagando apenas pelo tempo de execução, sem a necessidade de manter servidores ligados continuamente.

Etapa 2: 
- Amazon S3 + S3 Intelligent-Tiering

- Reduzir custos com armazenamento de arquivos, permitindo mover automaticamente dados pouco acessados para camadas mais baratas.

- A empresa pode armazenar relatórios, backups e documentos no Amazon S3, utilizando a funcionalidade de lifecycle para transferir arquivos antigos automaticamente para classes de armazenamento mais econômicas, como o S3 Glacier.

Etapa 3: 
- Amazon RDS (ou DynamoDB)

- Reduzir custos com gerenciamento e manutenção de bancos de dados, além de evitar desperdício de recursos com infraestrutura superdimensionada.

- A empresa pode migrar seu banco de dados para o Amazon RDS, utilizando instâncias ajustáveis conforme a demanda, ou optar pelo DynamoDB para um modelo serverless, reduzindo custos operacionais e pagando apenas pelo uso.



## Conclusão
A implementação das ferramentas AWS na empresa Abstergo Industries tem como expectativa a redução imediata de custos operacionais, maior escalabilidade dos serviços, otimização do armazenamento de dados e automação de processos internos. Esses benefícios contribuirão diretamente para o aumento da eficiência, da produtividade e da sustentabilidade tecnológica da empresa.

Recomenda-se a continuidade da utilização das ferramentas implementadas, bem como o monitoramento constante dos resultados obtidos, visando identificar novas oportunidades de melhoria e adoção de tecnologias que ampliem ainda mais a competitividade da organização.
